# RTOS Parking - STM32 Smart Parking System

RTOS Parking is an embedded smart parking system built with **STM32F401CCU6**, **STM32CubeIDE**, **STM32 HAL**, and **FreeRTOS/CMSIS-RTOS v2**. The project demonstrates real-time sensor monitoring, automatic gate control, parking slot status detection, visual indicators, buzzer alerts, and LCD status display.

## Features

- Two parking slot sensors using IR inputs.
- Entry, exit, and safety sensor detection.
- Automatic gate control using a PWM servo.
- Green/red LED status indicators.
- Buzzer alert when parking is full.
- 16x2 LCD display over I2C.
- FreeRTOS task-based system structure.

## Hardware and Tools

| Item | Description |
|---|---|
| MCU | STM32F401CCU6 |
| IDE | STM32CubeIDE |
| Configuration | STM32CubeMX |
| Firmware Package | STM32Cube FW_F4 V1.28.3 |
| RTOS | FreeRTOS with CMSIS-RTOS v2 |
| Driver Layer | STM32 HAL |
| Language | C |

## System Tasks

| Task | Purpose |
|---|---|
| `sensorTask` | Reads slot, entry, exit, and safety sensors. |
| `gateTask` | Controls gate opening and closing logic. |
| `indicatorTask` | Updates LEDs, buzzer, and LCD display. |
| `defaultTask` | Default CMSIS-RTOS task generated by CubeMX. |

## Pin Mapping

| Function | Pin | Mode |
|---|---|---|
| IR Slot 1 | PA0 | GPIO Input Pull-up |
| IR Slot 2 | PA1 | GPIO Input Pull-up |
| IR Entry | PA2 | GPIO Input Pull-up |
| IR Exit | PA3 | GPIO Input Pull-up |
| Servo Gate | PA5 | TIM2 PWM CH1 |
| Safety Sensor | PB2 | GPIO Input Pull-up |
| LCD SCL | PB6 | I2C1 SCL |
| LCD SDA | PB7 | I2C1 SDA |
| Buzzer | PB10 | GPIO Output |
| Red LED | PB12 | GPIO Output |
| Green LED | PB14 | GPIO Output |

## Build and Run

1. Open the project in **STM32CubeIDE**.
2. Ensure **STM32Cube FW_F4 V1.28.3** is installed.
3. Build the project from:

```text
Project > Build Project
```

4. Connect the board using ST-Link.
5. Flash and debug from:

```text
Run > Debug
```

## Behavior Summary

| Condition | System Response |
|---|---|
| Entry detected and parking is available | Gate opens. |
| Exit detected | Gate opens. |
| Both slots occupied | Parking is marked as full. |
| Parking full with vehicle request | Buzzer turns on. |
| Safety sensor active | Gate remains open. |
| No active request after delay | Gate closes automatically. |

## Notes

Build output folders such as `Debug/` and `Release/` are ignored to keep the repository clean. This repository is focused on the STM32CubeIDE implementation only.
